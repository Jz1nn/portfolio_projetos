Previsão de Vendas para Lojas Rossmann

Neste projeto, foi desenvolvido um modelo preditivo de Machine Learning para solucionar o desafio de previsão de vendas em 1.115 lojas da rede Rossmann. O objetivo era melhorar o planejamento financeiro e otimizar o processo de decisão estratégica da empresa.

O resultado final foi um modelo robusto capaz de prever as vendas das lojas para as próximas 6 semanas, integrando os resultados em uma API que permite consultas diretamente pelo aplicativo Telegram, facilitando o acesso em qualquer dispositivo conectado à internet.

As ferramentas utilizadas foram:

  • Python 3.8 e Pandas para análise de dados
  • Scikit-learn para desenvolvimento e avaliação de modelos de Machine Learning
  • Métricas de performance: RMSE, MAE, MAPE e R2
  • Modelos: Regressão Linear, Decision Tree Regressor, Random Forest Regressor e Gradient Boosting
  • API conectada ao Telegram para acesso remoto às previsões

Ensaio de Machine Learning

Nesse projeto, foram explorados os principais conceitos do treinamento, ajuste de parâmetros e limiares entre overfitting e underfitting de vários algoritmos de Machine Learning dentro das tarefas de classificação, regressão e agrupamento, através de um ensaio de Machine Learning.

Análise de Sentimento de Avaliações de Clientes com o VADER e Streamlit

Neste projeto, foi desenvolvido um sistema de análise de sentimento utilizando Processamento de Linguagem Natural (PLN) para analisar e classificar feedbacks de clientes. Utilizando o algoritmo VADER (Valence Aware Dictionary and sEntiment Reasoner) da biblioteca nltk, foi possível determinar automaticamente se o feedback de um cliente possui conotação positiva, negativa ou neutra.

O resultado final foi uma aplicação funcional hospedada na nuvem, acessível por qualquer dispositivo com conexão à internet, permitindo a interação direta dos usuários com o sistema para análise de feedback.

As ferramentas utilizadas foram:

  • Python, Pandas, NLTK.
  • Jupyter Notebook.
  • Terminal.
  • Streamlit para desenvolvimento e deploy do web app.
  • Github.
